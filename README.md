# Manhattan_Project
An AI-driven genealogical intelligence suite for the discovery and verification of lineage in Colonial American archival records.
The Manhattan Project was named due to the original project’s compartmentalized nature and our vision to create a compartmentalized LLM specifically designed for genealogical research.
The Manhattan Project is a compartmentalized AI intelligence suite for genealogists. By utilizing modular ‘agentic’ workflows, it performs high-precision transcription and verification of Colonial American primary sources. Just as its namesake was a coordinated effort to solve a monumental problem, this project aims to solve the problem of fragmented family history—uniting individuals through the power of verified data and historical transparency.
